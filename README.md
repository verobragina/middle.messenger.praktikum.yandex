### Usage
1. Installing modules
```
npm install
```
2. Building Project
```
npm run build
```
3. Runnig Project Local
```
npm run dev
```
3. Running Developer Server (localhost:3000)
```
npm run start
```
### Project Stack
**Building Tool:** Parcel

**Template System:** Handlebars.js

**Preprocessor:** Sass

**Server:** Express.js
